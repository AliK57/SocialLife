# SocialLife

Ein Übungsprojekt, um Vue 3 Schritt für Schritt zu lernen. Wir bauen eine Social-Media-App
und erweitern sie nach und nach.

## Projektstruktur (Kurzüberblick)

- `index.html` → Einstieg im Browser, lädt die App.
- `src/main.js` → startet Vue und mountet die App.
- `src/App.vue` → erste Komponente mit Template, Daten und Logik.
- `src/style.css` → globale Styles.
- `vite.config.js` → Vite-Konfiguration.

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Danach zeigt dir Vite eine URL an, die du im Browser öffnen kannst.
